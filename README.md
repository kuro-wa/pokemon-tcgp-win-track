# Pokemon TCGP Win Track

Pokemon TCGP (Trading Card Game Pocket) の対戦成績を記録し、勝敗、連勝数、ポイント推移を追跡する Web アプリケーションです。

## 技術スタック

- TypeScript
- React
- Vite
- Vitest
- LocalStorage

## セットアップ

```bash
npm install
```

## 開発コマンド

```bash
npm run dev
npm run build
npm run preview
```

## テスト

すべてのテストを実行します。

```bash
npx vitest run
```

単体テストのみ実行します。

```bash
npx vitest run tests/unit
```

結合テストのみ実行します。

```bash
npx vitest run tests/integration
```
